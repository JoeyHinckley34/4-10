# 4=10
Code repository for a fun problem <br>
# Problem Statement
Problem: give n numbers, what combination of + - * / and () makes x <br>
Base case: n = 4, x = 10 <br>
Example (7 + 8 + 5) / 2 = 10<br>
<br>
